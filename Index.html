DEFINICION

Podemos definir HTML5 como un estandar que sirve para definir la estructura y el contenido de una pagina Web. Es un estandar que sirve como referencia del software que conecta con la elaboracion de paginas web en sus diferentes versiones, define una estructura basica y un codigo (denominado HTML)para la definicion de contenido de una pagina web, como texto, imagenes, videos, juegos, entre otros. HTML5 especifica dos variantes de sintaxis para HTML, una version clasica y una variante XHTML conocida como sintaxis XHTML5 que debera servirse como sintaxis XML. Más informacion.

APLICACIONES

HTML proporciona un marco en el que los desarrolladores pueden poner su codigo y una aplicacion que lo convierte en algo que podemos ver y con lo que podemos interactuar. CSS permite diseñar como se veran los bits individuales de codigo una vez que se muestren en pantalla y JavaScript es lo que hace de todo algo interactivo, permitiendo hacer casi cualquier cosa.
